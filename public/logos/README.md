# Project logos

Drop a logo here named after the project and the app picks it up automatically.

- Filename = the project name lowercased with everything except letters/numbers
  stripped: `Nado` → `nado.png`, `Might&Magic` → `mightmagic.svg`,
  `Camp Network` → `campnetwork.png`.
- `.png` is checked first, then `.svg`.
- No file → the card shows a generated monogram tile instead.
- You can also set an explicit path or full URL per raid via the raid's
  `logo` field in an exported/imported backup.

The `.svg` files in here are generated placeholders — replace them with the
real project marks whenever you like.
